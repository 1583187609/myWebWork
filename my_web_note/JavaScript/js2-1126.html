1、认识DOM

DOM：文档对象模型（Document Object Model，简称DOM），是W3C组织推荐的处理可扩展标志语言的标准编程接口。常用节点：
(1)整个文档是一个文档节点；
(2)每个 HTML元素是元素节点；
(3)HTML元素内的文本是文本节点；
(4)每个 HTML属性是属性节点；
(5)注释是注释节点。
      DOM是javascript操作网页的接口，全称为文档对象模型(Document Object Model)。它的作用是将网页转为一个javascript对象，从而可以使用javascript对网页进行各种操作(比如增删内容)。浏览器会根据DOM模型，将HTML文档解析成一系列的节点，再由这些节点组成一个树状结构。DOM的最小组成单位叫做节点(node)。

DOM节点类型：节点至少拥有nodeType、nodeName和nodeValue这三个基本属性。

DOM节点的12种类型
节点类型数值说明
元素节点1element对应网页的HTML标签元素
属性节点2attribute对应网页中HTML标签的属性
文本节点3text代表网页中的HTML标签内容
CDATA节点4CDATASection类型只针对基于XML的文档，只出现在XML文档中，表示的是CDATA区域
实体引用名称节点5实体是一个声明，指定了在XML中取代内容或标记而使用的名称
实体名称节点6实体包含两个部分
处理指令节点7ProcessingInstruction
注释节点8comment表示网页中的HTML注释
文档节点9document表示HTML文档，也称为根节点，指向document对象
文档类型节点10DocumentType包含着与文档的doctype有关的所有信息
文档片段节点11在文档中没有对应的标记，是一种轻量级的文档，可以包含和控制节点，但不会像完整的文档那样占用额外的资源。
DTD声明节点12notation代表DTD中声明的符号

2、节点操作

获取元素：
document.getElementByid() //返回值是元素或者null
document.getElementsByTagName()//返回值是数组
document.getElementsByName()// 返回值是数组
document.getElementsByClassName()// 返回值是数组
document.querySelector('选择器')与document.querySelectorAll('选择器')
注: 获取的元素是已经创建的对象(节点对象)。querySelector()方法仅仅返回匹配指定选择器的第一个元素。如果你需要返回所有的元素，请使用 querySelectorAll() 方法替代。

创建元素：
var newElement=document.createElement("标签名")
      //返回值是一个新的节点对象,是内存中产生.不会直接存在于文档树,要主动添加进dom树才会显示

添加新元素到节点树：
xxNode.appendChild(newNode)
注意:调用者是文档树已经有的节点,才会加入到文档树,不然的话就只是存在于内存中

元素的属性：
xxx.className //可以有多个类名,但是要用空格隔开 xxx.className+=" xx"
xxx.classList //xxx.classList.add("xxxx1")
