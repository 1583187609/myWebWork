  1. 问：什么是字面量？
    答：是变量的字符串表示形式。它不是一种值，而是一种变量记法。例如：
    var test="hello world!";
    "hello world!"就是字符串字面量，test是变量名。
  2. 问：dom操作是异步的吗?
    答：dom的操作其实是同步的，但是渲染是异步的。因为JavaScript引擎线程跟GUI渲染线程是互斥的，即我执行的时候，你就靠边站，我执行完你才能执行。
